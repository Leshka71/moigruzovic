# 🚛 Мой Грузовик

Приложение для учёта доходов и расходов дальнобойщика в **Radmir CRMP**.

## 📦 Что умеет
- 🚛 Записывать рейсы и считать прибыль
- ⛽ Учитывать расходы на бензин
- 🔧 Фиксировать ремонты (СТО, ТК, Мойка, Штраф)
- 📊 Статистика по дням, неделям и месяцам
- 📅 История с фильтрами по датам
- 💾 Данные хранятся локально на твоём компьютере

## 📥 Скачать
Перейди в раздел [Releases](https://github.com/Leshka71/moigruzovic/releases) и скачай последнюю версию.

- Windows 10 / 11
- 500 МБ свободного места
