# 🚛 Мой Грузовик

> Приложение для учёта доходов и расходов дальнобойщика в **Radmir CRMP**

---

## 📦 Возможности
- 🚛 Записывай рейсы и отслеживай прибыль за день, неделю и месяц
- ⛽ Учитывай расходы на бензин
- 🔧 Фиксируй ремонты — СТО, ТК, Мойка, Штрафы
- 📊 Подробная статистика — всего рейсов, средний чек, прибыль по месяцам
- 📅 История с группировкой по дням и фильтрами по датам
- 💾 Данные хранятся локально на твоём компьютере
- 🔄 Авто-обновление при выходе новых версий
